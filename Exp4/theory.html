Generate the Multiplication Table of a Number Using Loops

Experiment 4: Generate the Multiplication Table of a Number Using Loops

Introduction

Programming often requires performing the same operation multiple times. Writing separate statements for each repetition makes programs lengthy and difficult to maintain. To overcome this problem, programming languages provide looping constructs that allow a block of code to execute repeatedly. In C programming, loops are widely used for repetitive tasks such as calculations, data processing, pattern generation, and table creation.

A multiplication table is a sequence of multiples of a given number. Generating a multiplication table is a common programming exercise that helps students understand loops, arithmetic operations, and program flow. This experiment uses a loop to calculate and display the multiplication table of a user-entered number.

Objectives

  • To understand the concept of loops in C programming.
  • To learn the structure and working of the for loop.
  • To generate the multiplication table of a number.
  • To perform arithmetic operations within a loop.
  • To develop logical thinking and programming skills.
  • To understand iterative execution of statements.

Theory

A loop is a control structure that executes a block of statements repeatedly as long as a specified condition remains true. Loops reduce code redundancy, improve readability, and increase program efficiency. C programming supports three types of loops:

  • for loop
  • while loop
  • do-while loop

The for loop is particularly useful when the number of iterations is known beforehand. Since a multiplication table generally contains values from 1 to 10, the for loop is the most suitable choice.

Structure of a For Loop

for(initialization; condition; increment/decrement)
{
    // statements
}

The for loop consists of three parts:

  • Initialization: Sets the starting value of the loop variable.
  • Condition: Determines whether the loop should continue.
  • Increment/Decrement: Updates the loop variable after each iteration.

In this experiment, the loop variable starts from 1 and continues until it reaches 10. During each iteration, the entered number is multiplied by the current value of the loop variable and displayed as part of the multiplication table.

Example

If the user enters the number 5, the program generates: 

5 × 1 = 5
5 × 2 = 10
5 × 3 = 15
5 × 4 = 20
5 × 5 = 25
5 × 6 = 30
5 × 7 = 35
5 × 8 = 40
5 × 9 = 45
5 × 10 = 50

This process demonstrates how loops automate repetitive calculations, reducing the need for multiple print statements.

Algorithm

  1. Start the program.
  2. Accept a number from the user.
  3. Initialize the loop variable i = 1.
  4. Check whether i is less than or equal to 10.
  5. Multiply the entered number by i.
  6. Display the multiplication result.
  7. Increment i by 1.
  8. Repeat steps 4–7 until i becomes greater than 10.
  9. Stop the program.
