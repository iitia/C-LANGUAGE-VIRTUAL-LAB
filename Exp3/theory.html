Determine Whether a Number is Even or Odd Using If-Else Statement

Experiment 3: Determine Whether a Number is Even or Odd Using If-Else Statement

Introduction

Conditional statements are used in C programming to make decisions based on specific conditions. One of the simplest applications of decision-making is determining whether a number is even or odd. This experiment uses the if-else statement along with the modulus (%) operator to classify a number.

Objective

  • To understand the use of conditional statements in C.
  • To learn the working of the modulus (%) operator.
  • To determine whether a number is even or odd.
  • To develop logical problem-solving skills.
  • To implement decision-making using if-else statements.

Theory

A number is considered even if it is completely divisible by 2 and leaves a remainder of 0. A number is considered odd if it leaves a remainder of 1 when divided by 2.

In C programming, the modulus operator (%) is used to obtain the remainder after division. The program checks the condition: 

if(num % 2 == 0)
    printf("Even");
else
    printf("Odd");

If the remainder is 0, the number is even; otherwise, it is odd. This concept is widely used in mathematical computations and logical decision-making applications.

Algorithm

  1. Start the program.
  2. Input a number from the user.
  3. Calculate the remainder using num % 2.
  4. If the remainder is 0, display "Even Number".
  5. Otherwise, display "Odd Number".
  6. Stop the program.
