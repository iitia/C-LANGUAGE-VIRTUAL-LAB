Determine Whether a Number is Even or Odd Using If-Else Statemen

Process Concept

Introduction

In C programming, decision-making is performed using conditional statements. The if-else statement allows the program to execute different blocks of code based on whether a condition is true or false. One of the simplest applications of this concept is checking whether a number is even or odd.

Objective

  • To understand conditional statements in C programming.
  • To use the modulus (%) operator for checking divisibility.
  • To determine whether a given number is even or odd.
  • To implement decision-making logic using if-else.
  • To improve problem-solving skills in programming.

Theory

A number is called even if it is divisible by 2 and leaves a remainder of 0. A number is called odd if it leaves a remainder of 1 when divided by 2.

In C programming, the modulus operator (%) is used to find the remainder of a division. By checking the condition num % 2 == 0, we can determine whether a number is even or odd.

Logic Used

  • If (num % 2 == 0), the number is Even.
  • Else, the number is Odd.

Algorithm

  1. Start the program.
  2. Input a number from the user.
  3. Check the condition num % 2 == 0.
  4. If true, display "Even Number".
  5. Otherwise, display "Odd Number".
  6. Stop the program.

C Programming Concepts Used

  • Variables and Data Types
  • Input and Output Functions (scanf, printf)
  • Modulus Operator (%)

  • Relational Operators

Applications

  • Number classification systems
  • Basic mathematical computations
  • Algorithm development practice
  • Educational programming exercises

Conclusion

This experiment demonstrates how decision-making can be implemented in C programming using the if-else statement. It helps students understand how to use logical conditions and arithmetic operators to solve simple real-world problems like checking even or odd numbers.
