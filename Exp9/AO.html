Write a program to store and display student information using structures.

This experiment demonstrates the concept of searching an element in an array using Linear Search technique in C programming. Linear search is a basic searching method where each element of the array is checked one by one until the required element is found or the array ends. In this experiment, the program starts from the first element of the array and compares each element with the target value. If a match is found, the position of the element is displayed. If the element is not found after checking all values, the program shows that the element is not present in the array. Linear search is simple to implement and works on both sorted and unsorted arrays. However, its time complexity is O(n), which makes it less efficient for large datasets. Overall, this experiment helps in understanding the basics of searching techniques and how sequential access works in arrays.
