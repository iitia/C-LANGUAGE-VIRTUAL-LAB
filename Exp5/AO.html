Write a program to find the factorial of a number using a user-defined function.

In this experiment, we will create a C program that calculates the factorial of a given number using a user-defined function. The function receives a number as input, computes its factorial, returns the result, and displays it. This experiment helps students understand functions, modular programming, and iterative calculations in C.
