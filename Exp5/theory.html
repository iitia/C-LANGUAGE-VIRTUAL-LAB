Factorial of a Number Using User-Defined Function

Introduction

Functions are one of the most important concepts in C programming. They allow programmers to divide a large program into smaller and reusable modules. A user-defined function is a function created by the programmer to perform a specific task.

Factorial Concept

The factorial of a non-negative integer n is the product of all positive integers from 1 to n. It is represented by n!. For example, 5! = 5 × 4 × 3 × 2 × 1 = 120.

Role of User-Defined Function

In this experiment, a user-defined function named factorial() is used to calculate the factorial of a given number. The function receives a number as input, performs the calculation, and returns the result to the main() function for display.

Programming Concepts Used

  • User-Defined Functions: Functions created by the programmer.
  • Function Call: Invokes the function to perform a task.
  • Return Statement: Sends the computed result back to the caller.
  • Loops: Used to repeatedly multiply numbers from 1 to n.
  • Variables: Store intermediate and final results.
