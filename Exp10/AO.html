Write a program to create, write, and read data from a text file.

In this experiment, we learned how to handle files in C programming. The program demonstrates how to create a text file, write data into it, and read the stored data back from the file. File handling allows data to be stored permanently in secondary memory, unlike variables that are lost after program execution. By using functions like fopen(), fprintf(), fscanf(), and fclose(), we can efficiently perform basic file operations. This experiment helps in understanding how real-world applications store and retrieve data using files.
