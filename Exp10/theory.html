Write a program to create, write, and read data from a text file

Introduction

File handling in C is used to store data permanently in files. Unlike variables stored in RAM, file data remains saved even after program execution ends.

File Operations

The main operations performed on files are creating, writing, reading, and closing files. These operations allow programs to handle large amounts of persistent data efficiently.

File Opening Modes

  • "r" → Open file for reading
  • "w" → Create a new file for writing (overwrite if exists)
  • "a" → Append data to an existing file

Important Functions

  • fopen() → Opens a file
  • fprintf() → Writes formatted data into a file
  • fscanf() → Reads formatted data from a file
  • fclose() → Closes the file

Working Principle

First, a file is created using fopen(). Data is written into the file using fprintf(). Later, the same data is read using fscanf(). Finally, the file is closed using fclose() to save all changes permanently.

Advantages

  • Provides permanent data storage
  • Useful for storing large data sets
  • Reduces memory usage during program execution

Conclusion

File handling is an essential concept in C programming that allows efficient storage and retrieval of data using external files.
