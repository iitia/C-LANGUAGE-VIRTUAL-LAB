C Language Architecture Explorer

In this experiment, students will explore the layered architecture of a c language, An interactive tool that visualizes the structure, compilation process, and execution flow of C programs. It helps students understand memory organization, function calls, and how C code is converted into an executable program. through an interactive simulator.
