Perform String Operations such as Length, Copy, and Concatenation

This experiment demonstrates basic string manipulation in C programming. Students learn how to find the length of a string, copy one string into another, and concatenate (join) two strings using built-in string handling functions. These operations are fundamental for processing textual data in C.
