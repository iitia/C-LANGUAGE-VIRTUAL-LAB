Calculate the Nth Fibonacci Number Using Recursion

Introduction

Recursion is a programming technique in which a function calls itself to solve a problem. It is useful for problems that can be divided into smaller subproblems. In this experiment, recursion is used to calculate the Nth Fibonacci number.

Fibonacci Series

The Fibonacci sequence is a series of numbers where each number is the sum of the two previous numbers. The sequence begins with 0 and 1.

Example: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

Recursive Formula

The Fibonacci number can be calculated recursively using the following formula:

fib(n) = fib(n-1) + fib(n-2)

Base Cases:
fib(0) = 0
fib(1) = 1

Working Principle

The recursive function repeatedly calls itself with smaller values of n until it reaches the base cases. The results are then combined to produce the Nth Fibonacci number.

Programming Concepts Used

  • Recursion: Function calling itself repeatedly.
  • Base Case: Stops recursive execution.
  • Function Calls: Executes recursive operations.
  • Return Values: Returns calculated Fibonacci numbers.
  • Problem Decomposition: Breaking a problem into smaller subproblems.
